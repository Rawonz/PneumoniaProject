PneumoScan - AI Chest X-Ray Analysis

Features
- VGG19-based pneumonia classification
- FastAPI backend
- React + Vite frontend
- Upload JPEG/PNG X-rays
- Confidence score
- Class probabilities
- Swagger API docs

Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

Frontend
npm install
npm run dev
